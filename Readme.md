# 🔐 Password Generator

A modern and responsive **Password Generator** built with **HTML, CSS, and JavaScript**. This application helps users create strong and secure passwords by selecting the desired password length and character types, including uppercase letters, lowercase letters, numbers, and special symbols.

---

## 🚀 Features

- 🔒 Generate strong and secure passwords
- 📏 Customize password length
- 🔠 Include Uppercase Letters (A-Z)
- 🔡 Include Lowercase Letters (a-z)
- 🔢 Include Numbers (0-9)
- ✨ Include Special Characters (!@#$%^&*)
- 📋 Copy generated password with one click
- 📱 Fully Responsive Design
- 🎨 Clean and Modern User Interface

---

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/chaudharysachinraj/Password-Generator.git
```

Go to the project folder:

```bash
cd Password-Generator
```

Open `index.html` in your browser.

---

## 🎯 How to Use

1. Enter the desired password length.
2. Select the character types:
   - Uppercase Letters
   - Lowercase Letters
   - Numbers
   - Symbols
3. Click the **Generate Password** button.
4. Copy the generated password using the **Copy** button.
